# Astra Analytics HTML/CSS Dashboard 📊

A modern and responsive analytics dashboard built using HTML, CSS, and JavaScript.  
This project focuses on creating a clean business dashboard interface with reusable UI sections, responsive design, and interactive elements.

## 🚀 Features

- Responsive dashboard layout
- Modern dark theme UI
- Sidebar navigation
- Dashboard statistics cards
- Analytics overview section
- Revenue and sales insights
- User management interface
- Products management page
- Reports section
- Settings page
- Interactive UI components

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap
- Font Awesome Icons
